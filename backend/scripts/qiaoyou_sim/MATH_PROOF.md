
## 九、关键计算公式汇总

### 9.1 增值税

```python
# 小规模纳税人（2025-12 ~ 2026-05）
销项税 = 含税销售额 / 1.01 × 0.01  # 减按1%征收
应交增值税 = 销项税（不抵扣进项）

# 一般纳税人（2026-06起）
销项税 = 含税销售额 / 1.06 × 0.06
应交增值税 = max(销项税 - 进项税 - 期初留抵, 0)
```

### 9.2 附加税

```python
# 城建税（不享受30万免征，只享受减半）
城建税 = 销项税 × 7% × 50% = 销项税 × 0.035

# 教育费附加、地方教育附加（季度≤30万免征）
if 季度销售额 <= 300000:
    教育费附加 = 0
    地方教育附加 = 0
else:
    教育费附加 = 销项税 × 3% × 50%
    地方教育附加 = 销项税 × 2% × 50%
```

### 9.3 所得税

```python
# 累计利润（与利润表口径一致）
累计利润 = 收入(6001+6051) - 成本(6401) - 费用(6601+6602+6603+6403明细) + 营业外收入(6301) - 营业外支出(6701)

# 小型微利企业实际税负 5%
应交所得税 = max(累计利润, 0) × 5%

# delta = 当期应交 - 已计提
delta = 应交所得税 - 已计提所得税(222105 贷方余额)
if delta > 0:
    计提：Dr 6801 / Cr 222105
else:
    冲回：Dr 222105 / Cr 6801
```

### 9.4 BS 勾稽

```
资产 = 货币资金(1001+1002) + 存货(1405) + 固定资产净值(1601-1602) + ...
负债 = 应付账款(2202) + 应交税费(222101+222103+222107+222110+222111+222112+222105) + ...
权益 = 本年利润(4103) + 未分配利润(4104) + 实收资本(4001) + ...

资产 = 负债 + 权益
```
