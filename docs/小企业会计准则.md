# 📖 小企业会计准则速查

> 开发用速查：计算公式 + 业务规则 + 税率表

---

## 一、核心计算公式

### 1.1 收入

```
销售收入 = 不含税售价 × 数量
含税售价 = 不含税售价 × (1 + 增值税税率)
应收账款 = 含税售价 - 现金折扣（如有）

一般纳税人：增值税 = 不含税收入 × 税率(13%/9%/6%)
小规模纳税人：增值税 = 不含税收入 × 征收率(1%)
```

### 1.2 成本

```
发出存货成本 = 发出数量 × 存货单价
期末存货成本 = 期初存货 + 本期入库 - 本期发出

先进先出法：按入库先后顺序计价
加权平均法：加权平均单价 = (月初金额 + 本月入库金额) ÷ (月初数量 + 本月入库数量)
```

### 1.3 固定资产折旧

```
年折旧额 = (原值 - 残值) ÷ 使用年限
月折旧额 = 年折旧额 ÷ 12

残值 = 原值 × 残值率（一般 5%）
```

**最低折旧年限**：

| 资产类别 | 最低年限 |
|---------|---------|
| 房屋、建筑物 | 20 年 |
| 机器、设备 | 10 年 |
| 器具、工具、家具 | 5 年 |
| 运输工具 | 4 年 |
| 电子设备 | 3 年 |

### 1.4 无形资产摊销

```
年摊销额 = 无形资产成本 ÷ 使用寿命（年）
月摊销额 = 年摊销额 ÷ 12

使用寿命不确定：摊销期不低于 10 年
```

### 1.5 长期待摊费用

```
月摊销额 = 长期待摊费用总额 ÷ 摊销期限（月）
```

## 二、负债计算

### 2.1 流动负债

```
短期借款利息 = 借款本金 × 年利率 × 借款天数 ÷ 360

应付账款 = 采购价款 + 增值税进项税额
预收账款 = 预收货款金额
应付职工薪酬 = 工资 + 奖金 + 社保（单位部分） + 公积金（单位部分）
应交税费 = 增值税 + 附加税 + 所得税 + 个人所得税
```

### 2.2 增值税

```
一般纳税人：
  应纳税额 = 销项税额 - 进项税额
  销项税额 = 不含税销售额 × 税率(13%/9%/6%)
  进项税额 = 取得专票上注明的税额

小规模纳税人：
  应纳税额 = 不含税销售额 × 征收率(1%)
  不含税销售额 = 含税销售额 ÷ (1 + 征收率)
```

### 2.3 附加税

```
城市维护建设税 = 增值税额 × 7%（市区）/ 5%（县城）/ 1%（其他）
教育费附加 = 增值税额 × 3%
地方教育附加 = 增值税额 × 2%

小微企业减半：乘以 50%
```

### 2.4 个人所得税（工资薪金）

```
应纳税所得额 = 工资收入 - 5000（起征点）- 专项扣除 - 专项附加扣除

税率表：
  ≤ 36,000        税率 3%   速算扣除数 0
  36,000-144,000   税率 10%  速算扣除数 2,520
  144,000-300,000  税率 20%  速算扣除数 16,920
  300,000-420,000  税率 25%  速算扣除数 31,920
  420,000-660,000  税率 30%  速算扣除数 52,920
  660,000-960,000  税率 35%  速算扣除数 85,920
  > 960,000        税率 45%  速算扣除数 181,920

应纳税额 = 应纳税所得额 × 税率 - 速算扣除数
```

## 三、利润计算

### 3.1 利润表公式

```
营业收入 = 主营业务收入 + 其他业务收入
营业成本 = 主营业务成本 + 其他业务成本

毛利润 = 营业收入 - 营业成本

营业利润 = 毛利润 - 营业税金及附加 - 销售费用 - 管理费用 - 财务费用 + 投资收益

利润总额 = 营业利润 + 营业外收入 - 营业外支出

净利润 = 利润总额 - 所得税费用
```

### 3.2 所得税费用

```
应纳税所得额 = 利润总额 ± 纳税调整

小微企业所得税（优惠至 2027 年底）：
  应纳税额 = 应纳税所得额 × 25% × 20% = 应纳税所得额 × 5%

  例：年利润 200 万元
  所得税 = 200 × 5% = 10 万元
```

### 3.3 小微企业判定条件（同时满足）

- 年应纳税所得额 ≤ 300 万元
- 从业人数 ≤ 300 人
- 资产总额 ≤ 5000 万元
- 从事国家非限制和禁止行业

## 四、现金流量

```
经营活动现金流入 = 销售商品收到的现金 + 收到的税费返还 + 收到其他与经营活动有关的现金
经营活动现金流出 = 购买商品支付的现金 + 支付给职工的现金 + 支付的各项税费 + 支付其他与经营活动有关的现金

经营活动现金流量净额 = 经营活动现金流入 - 经营活动现金流出

投资活动现金流量净额 = 投资活动现金流入 - 投资活动现金流出
筹资活动现金流量净额 = 筹资活动现金流入 - 筹资活动现金流出

现金净增加额 = 经营活动净额 + 投资活动净额 + 筹资活动净额
期末现金余额 = 期初现金余额 + 现金净增加额
```

## 六、常见会计分录

### 采购

```
一般纳税人：
借：原材料              10,000
借：应交税费-应交增值税(进项) 1,300
  贷：应付账款            11,300

小规模纳税人：
借：原材料              11,300
  贷：应付账款            11,300
```

### 销售

```
确认收入：
借：应收账款            22,600
  贷：主营业务收入        20,000
  贷：应交税费-应交增值税(销项) 2,600

结转成本：
借：主营业务成本        15,000
  贷：库存商品            15,000
```

### 折旧

```
借：管理费用-折旧        792
借：销售费用-折旧        396
  贷：累计折旧            1,188

计算：(50,000 - 2,500) ÷ 5 ÷ 12 = 791.67 ≈ 792
```

### 工资

```
计提：
借：管理费用-工资        50,000
借：销售费用-工资        30,000
  贷：应付职工薪酬        80,000

发放：
借：应付职工薪酬        80,000
  贷：银行存款            70,000
  贷：应交税费-个人所得税   5,000
  贷：其他应付款-社保      5,000
```

### 税金

```
计提附加税：
借：营业税金及附加      312
  贷：应交税费-城建税      182
  贷：应交税费-教育费附加   78
  贷：应交税费-地方教育附加  52

计提所得税：
借：所得税费用          5,000
  贷：应交税费-应交企业所得税 5,000
```
