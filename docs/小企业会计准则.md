# 📖 小企业会计准则速查

> **法规依据**: 财政部《小企业会计准则》（财会〔2011〕17号，2011-10-18 公布，2013-01-01 起施行）
> **时效性**: 现行有效
> **适用范围**: 在中华人民共和国境内依法设立的、符合《中小企业划型标准规定》所规定的小型企业标准的企业（金融机构、上市公司、企业集团内母子公司除外）
> **版本**: v2.0 | 2026-07-02

---

## 一、准则概述

### 1.1 准则结构

《小企业会计准则》分 **10 章 90 条**，外加附录（会计科目、主要账务处理和财务报表）：

| 章 | 内容 | 条数 |
|----|------|------|
| 第一章 | 总则 | 1-4 |
| 第二章 | 资产 | 5-47 |
| 第三章 | 负债 | 48-55 |
| 第四章 | 所有者权益 | 56-60 |
| 第五章 | 收入 | 61-65 |
| 第六章 | 费用 | 66-70 |
| 第七章 | 利润及利润分配 | 71-76 |
| 第八章 | 外币业务 | 77-83 |
| 第九章 | 财务报表 | 84-88 |
| 第十章 | 附则 | 89-90 |

### 1.2 与《企业会计准则》的主要区别

| 项目 | 小企业会计准则 | 企业会计准则 | 准则条文 |
|------|--------------|------------|---------|
| 计量属性 | 历史成本 | 历史成本 + 公允价值 | 第六条 |
| 资产减值 | 不计提 | 需计提 | 第六条 |
| 长期股权投资 | 成本法 | 成本法 / 权益法 | 第二十四条 |
| 借款费用 | 直接资本化 | 需区分专门/一般 | 第三十三条 |
| 所得税 | 应付税款法 | 资产负债表债务法 | 第七十三条 |
| 财务报表 | 4 表（简化） | 4 表 + 附注 | 第七十六条 |
| 坏账处理 | 实际发生时直接核销 | 需计提坏账准备 | 第十条 |

### 1.3 资产减值的核心规定（第六条）

> 小企业的资产应当按照成本计量，不计提资产减值准备。

实际损失发生时直接核销（计入营业外支出），不预提减值准备。包括：
- 不计提存货跌价准备
- 不计提固定资产减值准备
- 不计提无形资产减值准备
- 不计提坏账准备

## 三、资产

### 3.1 流动资产

#### 3.1.1 存货（第十一条至第十五条）

**定义**：小企业在日常生产经营过程中持有以备出售的产成品或商品、处在生产过程中的在产品、将在生产过程或提供劳务过程中耗用的材料和物料等。

**存货成本计量**（第十二条）：
- 外购存货成本 = 购买价款 + 相关税费 + 运输费 + 装卸费 + 保险费 + 其他直接费用
- 但**不含**可抵扣的增值税进项税额
- 投资者投入存货成本 = 评估价值
- 盘盈存货成本 = 同类或类似存货的市场价格或评估价值

**发出存货计价**（第十三条）：
- 应当采用 **先进先出法、加权平均法或者个别计价法**
- 计价方法一经选用，不得随意变更
- 本系统采用 **移动加权平均法**

```
移动加权平均法计算公式：
avg_cost = (old_value + new_cost) / (old_qty + new_qty)
出库时：unit_cost = inv.average_cost（锁定到 SaleItem.unit_cost）
```

**存货毁损**（第十五条）：
- 净损失计入 **营业外支出**（6701）
- 净收益计入 **营业外收入**（6301）

#### 3.1.2 应收及预付款项（第九条至第十条）

**坏账损失确认条件**（第十条）：
1. 债务人依法宣告破产、关闭、解散、被撤销
2. 债务人死亡或依法被宣告失踪、死亡
3. 债务人逾期 3 年以上未清偿，且有确凿证据证明已无力清偿
4. 与债务人达成债务重组协议或法院批准破产重整计划后无法追偿
5. 因自然灾害、战争等不可抗力导致无法收回
6. 国务院财政、税务主管部门规定的其他条件

> **关键**：应收及预付款项的坏账损失应当于 **实际发生时** 计入营业外支出，同时冲减应收及预付款项。**不计提坏账准备**。

### 3.2 非流动资产

#### 3.2.1 固定资产（第二十七条至第三十四条）

**定义**：小企业为生产产品、提供劳务、出租或经营管理而持有的，使用寿命超过 1 年的有形资产。

**成本计量**（第二十八条）：
- 外购固定资产成本 = 购买价款 + 相关税费 + 运输费 + 装卸费 + 保险费 + 安装费
- **不含**可抵扣的增值税进项税额（一般纳税人专票）
- 小规模纳税人或普票：按价税合计计入原值

**折旧**（第二十九条至第三十一条）：
- **折旧方法**：年限平均法（直线法），技术进步确需加速的可采用双倍余额递减法或年数总和法
- **折旧公式**：月折旧额 = 原值 × (1 - 残值率) ÷ 使用年限 ÷ 12
- **累计折旧上限** = 原值 × (1 - 残值率)
- **当月增加当月不计提，下月起计提**；当月减少当月仍计提，下月起不提（第三十一条）

```
系统实现：
- engine_fixed_asset.calculate_depreciation_monthly()
- 月折旧额 = 原值 × (1 - 残值率) ÷ 使用年限 ÷ 12
- 借：管理费用(6601) / 销售费用(6602)
- 贷：累计折旧(1602)
```

**处置**（第三十四条）：
- 处置收入 - 账面价值 - 相关税费 - 清理费用 = 净额
- 净额 > 0：计入 **营业外收入**（6301）
- 净额 < 0：计入 **营业外支出**（6701）
- **现金流归类**：投资活动现金流入

#### 3.2.2 无形资产（第三十八条至第四十五条）

**定义**：小企业为生产产品、提供劳务、出租或经营管理而持有的、没有实物形态的可辨认非货币性资产，包括土地使用权、专利权、商标权、著作权、非专利技术等。

**摊销规则**（与固定资产相反）：
- **当月增加的无形资产，当月开始摊销**
- **处置当月不再摊销**
- 摊销方法：年限平均法，一般无残值
- 月摊销额 = 原值 ÷ 摊销年限 ÷ 12

> **⚠️ 关键区别**：无形资产"当月增当月摊"，固定资产"当月增下月提"。两者规则相反，不能混淆。

### 3.3 长期待摊费用（第五十二条）

**租入固定资产改良支出**：
- 发生装修费：计入长期待摊费用（1801，资产类）
- 按租期平均摊销：每期管理费用

## 四、负债

### 4.1 应付账款（第四十九条）

- 因购买商品、接受劳务等日常生产经营活动应支付的款项
- 应当按照发生额入账
- 实际发生时：借记"库存商品"等，贷记本科目
- 付款时：借记本科目，贷记"银行存款"

### 4.2 预收账款（第五十条）

- 小企业按照合同规定预收的款项
- **预收的钱是负债（欠客户的货），不是收入**
- 收款时：借记"银行存款"，贷记本科目
- 发货确认收入时：借记本科目，贷记"主营业务收入"+"应交税费-应交增值税"

> **关键因果**：钱先到 ≠ 收入先确认。预收款的因是收款，收入的因是发货（权责发生制）。

### 4.3 应付职工薪酬（第五十一条）

- 计提时：借记"管理费用/销售费用"，贷记本科目
- 发放时：借记本科目，贷记"银行存款"+"应交税费-个人所得税"
- **工资费用在"计提"时确认，不在"发放"时确认**
- 个税是代扣代缴，实发时才产生纳税义务

### 4.4 应交税费

#### 4.4.1 增值税

**小规模纳税人**：
- 征收率：3%（销售货物、提供劳务）
- 政策优惠：季度销售额 ≤ 30 万元，普通发票免征增值税（财税公告 2023 年第 12 号，2023.1.1-2027.12.31）
- **专票不免税**：开具专用发票必须按征收率征收
- 销项税额 = 含税销售额 ÷ (1 + 征收率) × 征收率

```
系统实现（小规模销项税计算）：
1. 季度累计不含税销售额 = 本笔前累计 + 本笔
2. 累计 ≤ 30 万 + 普票：免征
3. 累计 > 30 万 或 专票：按 tax_rate 征收
```

**一般纳税人**：
- 税率：13%（货物）、9%（农产品、交通运输等）、6%（现代服务）
- 销项税额 = 不含税销售额 × 税率
- 进项税额 = 已认证专用发票税额
- 应纳增值税 = 销项税额 - 进项税额（留抵下期）

#### 4.4.2 附加税

- **城建税**：实际缴纳的增值税 × 7%（市区）、5%（县城）、1%（其他）
- **教育费附加**：实际缴纳的增值税 × 3%
- **地方教育附加**：实际缴纳的增值税 × 2%
- 合计附加税率：12%（市区）
- **小规模减半**：6%（市区）

```
入账：
借：税金及附加(6403)
贷：应交附加税(222104)
```

> **关键因果**：附加税是增值税的衍生，增值税的因是发票。附加税是"衍生的衍生"。

#### 4.4.3 企业所得税

**小型微利企业优惠**（财税公告 2023 年第 12 号，2023.1.1-2027.12.31）：

| 条件 | 税率 | 实际税负 |
|------|------|---------|
| 年应纳税所得额 ≤ 300 万元 | 减按 25% 计入 × 20% 税率 | **5%** |
| 年应纳税所得额 > 300 万元 | 法定税率 25% | 25% |
| 个人实体（entity_type=personal） | 不缴企业所得税 | 0% |

```
系统实现（AccountingEngine.calculate_income_tax）：
if entity_type == 'personal':
    tax_payable = 0
elif profit < 0:
    tax_payable = 0  # 亏损不缴税
elif small_micro and profit <= 3,000,000:
    tax_payable = profit × 0.25 × 0.20  # = profit × 5%
else:
    tax_payable = profit × 0.25  # 法定税率
```

## 五、收入

### 5.1 收入确认原则（第五十九条至第六十条）

**第五十九条**：销售商品收入应当在 **同时满足** 下列条件时确认：
1. 已将商品所有权上的主要风险和报酬转移给购货方
2. 既没有保留与所有权相联系的继续管理权，也没有对已售出的商品实施控制
3. 相关的经济利益很可能流入企业
4. 相关的收入和成本能够可靠地计量

**不同销售方式的收入确认时点**：

| 销售方式 | 收入确认时点 | 准则条文 |
|---------|------------|---------|
| 现场销售（当场收款） | 交易完成时 | 第五十九条 |
| 账期销售（先发货后收款） | **发货时**确认收入 | 第五十九条 |
| 预收销售（先收款后发货） | **发货时**确认收入（不是收款时） | 第五十九条 |
| 分期收款销售 | **合同约定收款日**分期确认 | 第六十条 |

> **关键因果**：
> - 不开发票 ≠ 不确认收入 ≠ 不交税。收入的因是"交易完成"，发票只是税务凭据。
> - 钱先到 ≠ 收入先确认（预收）。货先发 ≠ 收入先确认（分期收款）。
> - 这是"权责发生制"——权利义务发生即确认，不管钱到没到。

### 5.2 未开票销售的处理

本系统允许未开票销售（与准则一致）：
- **未开票只确认收入**（无销项税）
- 后期补开发票时补记销项税
- 凭证日期 = 发票日期
- 系统使用 1131（应收-未开票）记录此类销售

> **⚠️ BS 应收账款必须包含 1131**：`accounts_receivable = _balance("1122") + _balance("1131")`
> 未开票销售场景下，1131 记录销售收入但未开票；若 BS 只读 1122 会导致 A-(L+E)≠0。

### 5.3 视同销售（第六十二条）

小企业发生非货币性资产交换、将货物用于集体福利或利润分配等视同销售行为，应当 **按公允价值确认收入**：

| 情形 | 借方 | 贷方 |
|------|------|------|
| 自产货物用于职工福利 | 应付职工薪酬 | 主营业务收入 + 应交增值税(销项) |
| 货物无偿赠送 | 营业外支出 | 主营业务收入 + 应交增值税(销项) |
| 同时结转成本 | 主营业务成本 | 库存商品 |

## 六、费用

### 6.1 费用确认（第六十六条至第七十条）

**期间费用**（本系统 6601-6603）：
- **管理费用（6601）**：行政管理部门发生的费用
- **销售费用（6602）**：销售商品过程中发生的费用
- **财务费用（6603）**：利息支出、手续费、现金折扣

**税金及附加（6403）**：
- 包括消费税、城建税、教育费附加、资源税、土地增值税、房产税、车船税、印花税等
- **不含增值税**（增值税是价外税，不通过损益科目核算）

**营业成本（6401）**：
- 已售存货的成本结转
- 必须使用 **SaleItem.unit_cost**（锁定移动加权平均成本）
- 禁止使用 Product.purchase_price

### 6.2 营业外收支（第七十四条）

**营业外收入（6301）**：
- 违约金收入、罚款收入、捐赠收入、债务重组利得
- 盘盈收益、固定资产处置净收益

**营业外支出（6701）**：
- 罚款支出、捐赠支出、非常损失
- 坏账实际核销
- 固定资产处置净损失

## 七、利润及利润分配

### 7.1 利润计算（第七十一条）

```
利润总额 = 营业收入 - 营业成本 - 税金及附加 - 销售费用 - 管理费用 - 财务费用
         + 投资收益 + 其他业务收入 + 资产处置收益
         + 营业外收入 - 营业外支出 - 资产处置损失

净利润 = 利润总额 - 所得税费用
```

**系统计算口径**：
- 营业收入：6001（主营业务收入）+ 6051（其他业务收入）
- 营业成本：6401（主营业务成本）
- 期间费用：6601（管理费用）+ 6602（销售费用）+ 6603（财务费用）
- 税金及附加：6403（含各明细 640301-640311）
- 投资收益：5111
- 资产处置：6111（收益）/ 6711（损失）
- 营业外：6301（收入）/ 6701（支出）
- 所得税：6801

### 7.2 所得税计算（第七十三条）

本系统采用 **应付税款法**（不确认递延所得税）：

```
应纳税所得额 = 利润总额 ± 纳税调整（小企业通常无调整）
所得税费用 = 应纳税所得额 × 适用税率

小型微利企业优惠（2023-2027）：
  年应纳税所得额 ≤ 300 万元：减按 25% 计入 × 20% 税率 = 实际 5%
  年应纳税所得额 > 300 万元：法定 25%

亏损不缴所得税（profit < 0 时 tax_payable = 0）
个人实体不缴企业所得税（entity_type=personal 时 tax_payable = 0）
```

### 7.3 利润分配（第七十五条至第七十六条）

```
1. 提取法定盈余公积：净利润 × 10%（累计达注册资本 50% 可不再提取）
   借：利润分配-提取法定盈余公积
   贷：盈余公积

2. 向投资者分配利润
   借：利润分配-应付利润
   贷：应付利润
```

## 八、财务报表

### 8.1 报表组成（第七十六条）

小企业财务报表包括 **4 表**（简化版，无附注）：

| 报表 | 性质 | 平衡关系 |
|------|------|---------|
| 资产负债表 | 时点报表 | 资产 = 负债 + 所有者权益 |
| 利润表 | 期间报表 | 收入 - 成本费用 = 利润 |
| 现金流量表 | 期间报表 | 现金流入 - 现金流出 = 净增加 |
| 附注 | 补充说明 | 文字说明 |

### 8.2 资产负债表核心恒等式

```
资产总额 = 负债总额 + 所有者权益总额

系统校验：
- 应收账款 = 1122 借方余额 + 1131 借方余额（未开票销售）
- 应付税款 = 222107 贷方余额 + 222103 贷方余额（小规模）
- 库存价值 = Inventory.total_value = Σ(StockMove.total_cost WHERE quantity>0) - Σ(StockMove.total_cost WHERE quantity<0)
```

### 8.3 现金流量表分类

| 分类 | 内容 | 典型科目 |
|------|------|---------|
| 经营活动 | 销售商品、购买商品、支付工资、交税 | 1001、1002、1122、1405、2202、2211 |
| 投资活动 | 购建处置固定资产、无形资产 | 1601、1701、6111、6711 |
| 筹资活动 | 借款、还款、分配利润 | 2001、2501、4104 |

> **关键**：处置固定资产的现金流必须归"投资活动"，不能归"经营活动"。

## 九、关键因果链总结

### 9.1 三税种关系

```
发票（销项/进项）→ 增值税 → 附加税 → 入账（6403/222104）
                                        ↓
收入(发票) - 成本(出库) - 费用(费用单+折旧+摊销+附加税) + 营外收支 → 利润
                                        ↓
                                   所得税 → 入账（6801/222105）
```

| 税种 | 因 | 因的类型 |
|------|----|---------|
| 增值税 | 发票 + 视同销售事件 | 外部输入 |
| 附加税 | 增值税 | 衍生数字 |
| 所得税 | 利润 → 收入(发票) - 成本(出库) - 费用(费用单) | 衍生数字的衍生数字 |

### 9.2 核心原则

> 1. 所有报表数字都是"果"，它们的"因"是凭证，凭证的"因"是外部输入。**报表不能反过来当源**。
> 2. 不开发票 ≠ 不确认收入 ≠ 不交税。
> 3. 钱先到 ≠ 收入先确认（预收）。货先发 ≠ 收入先确认（分期收款）。
> 4. 小企业不计提减值，损失实际发生时才入账。
> 5. 无形资产"当月增当月摊"，固定资产"当月增下月提"。两者规则相反。
