## 二、核心计算公式（系统实现）

### 2.1 资产负债表

**依据**: 小企业会计准则第二十一条 — 资产 = 负债 + 所有者权益

```
【资产】
货币资金 = 银行余额 + 现金余额
应收账款 = 未收款销售单总额
存货 = 库存数量 × 采购单价
固定资产净值 = 原值 - 累计折旧
无形资产净值 = 原值 - 累计摊销
流动资产合计 = 货币资金 + 应收账款 + 存货
非流动资产合计 = 固定资产净值 + 无形资产净值
资产总计 = 流动资产 + 非流动资产

【负债】
应付账款 = 未付款采购单 + 未付费用 + 未付固定资产 + 未付存货
应交税费 = 销项税额 - 已认证进项税额（≥0）
长期借款 = 期初余额录入
负债合计 = 应付账款 + 应交税费 + 长期借款

【所有者权益】
实收资本 = 期初余额录入
未分配利润 = 期初未分配 + 本期收入 - 本期成本 - 本期费用 - 折旧 - 摊销 - 所得税
所有者权益合计 = 实收资本 + 未分配利润

【校验】
资产总计 = 负债合计 + 所有者权益合计（容差 ±0.01）
```

### 2.2 利润表

**依据**: 小企业会计准则第七十七条

```
营业收入 = 已完成销售单总额（含税）
营业成本 = 销售数量 × 采购单价
毛利润 = 营业收入 - 营业成本

销售费用 = Expense WHERE functional_category='销售费用'
管理费用 = Expense WHERE functional_category='管理费用'
财务费用 = Expense WHERE functional_category='财务费用'
营业税金及附加 = 增值税附加税

营业利润 = 毛利润 - 税金及附加 - 销售费用 - 管理费用 - 财务费用
利润总额 = 营业利润（营业外收支暂为 0）
所得税费用 = 利润总额 × 5%（小微企业）或 25%（一般企业）
净利润 = 利润总额 - 所得税费用
```

### 2.3 增值税

**依据**: 增值税暂行条例 + 财政部 税务总局公告 2023 年第 19 号（执行至 2027 年底）

```
【小规模纳税人】
法定征收率 = 3%
优惠征收率 = 1%（2023-2027）
月销售额 ≤ 10 万元：免征增值税

不含税销售额 = 含税销售额 ÷ (1 + 1%)
应纳增值税 = 不含税销售额 × 1%

附加税（小微企业减半）：
  城建税 = 增值税 × 7% × 50%
  教育费附加 = 增值税 × 3% × 50%
  地方教育附加 = 增值税 × 2% × 50%
  月销售额 ≤ 10 万：教育费附加 + 地方教育附加免征

【一般纳税人】
税率：13%（货物）/ 9%（运输、建筑）/ 6%（服务）
应纳增值税 = 销项税额 - 进项税额
销项税额 = 不含税销售额 × 税率
进项税额 = 已认证专票税额

附加税（无减半）：
  城建税 = 增值税 × 7%
  教育费附加 = 增值税 × 3%
  地方教育附加 = 增值税 × 2%
```

### 2.4 企业所得税

**依据**: 企业所得税法第五条 + 财政部 税务总局公告 2023 年第 12 号（执行至 2027 年底）

```
【小微企业】（同时满足：利润 ≤ 300 万，人数 ≤ 300，资产 ≤ 5000 万）
应纳税额 = 利润总额 × 25% × 20% = 利润总额 × 5%

【一般企业】
应纳税额 = 利润总额 × 25%

【高新技术企业】
应纳税额 = 利润总额 × 15%
```

### 2.5 固定资产折旧

**依据**: 小企业会计准则第三十条 + 企业所得税法实施条例第六十条

```
【年限平均法（直线法）】— 默认
年折旧额 = (原值 - 残值) ÷ 使用年限
月折旧额 = 年折旧额 ÷ 12

【双倍余额递减法】
年折旧率 = 2 ÷ 使用年限
年折旧额 = 账面净值 × 年折旧率
最后 2 年改直线法

【年数总和法】
年数总和 = n × (n + 1) ÷ 2
年折旧额 = (原值 - 残值) × 剩余年限 ÷ 年数总和

残值 = 原值 × 残值率（一般 5%）
折旧时点：当月增加下月提，当月减少下月停
```

**最低折旧年限**（企业所得税法实施条例第六十条）：

| 资产类别 | 最低年限 |
|---------|---------|
| 房屋、建筑物 | 20 年 |
| 机器设备 | 10 年 |
| 器具工具家具 | 5 年 |
| 运输工具 | 4 年 |
| 电子设备 | 3 年 |

### 2.6 无形资产摊销

**依据**: 小企业会计准则第四十一条

```
年摊销额 = 成本 ÷ 使用寿命（年）
月摊销额 = 年摊销额 ÷ 12

使用寿命不确定：摊销期不低于 10 年
```

### 2.7 发票金额计算

**依据**: 小企业会计准则第十五条

```
不含税金额 = 含税金额 ÷ (1 + 税率)
税额 = 含税金额 - 不含税金额
```

## 五、会计恒等式校验

**依据**: 小企业会计准则第二十一条

```
资产负债表校验：
  资产总计 = 负债合计 + 所有者权益合计（容差 ±0.01）

期初余额校验：
  资产 = 负债 + 实收资本 + 未分配利润

利润表校验：
  净利润 = 利润总额 - 所得税费用

系统实现：
  GET /api/accounting-check/balance-sheet  → 返回校验结果 + ai_instruction
  GET /api/accounting-check/income-statement → 返回校验结果
  GET /api/accounting-check/vat → 返回增值税计算校验
  GET /api/accounting-check/income-tax → 返回所得税计算校验
```
