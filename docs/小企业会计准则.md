## 一、准则概述

**法规依据**：财政部财会〔2011〕17 号《小企业会计准则》

**适用范围**：
- 符合《中小企业划型标准规定》（工信部联企业〔2011〕300 号）
- 本系统面向的 SMB 用户均适用

**与《企业会计准则》的主要区别**：

| 项目 | 小企业会计准则 | 企业会计准则 |
|------|--------------|------------|
| 计量属性 | 历史成本（不使用公允价值） | 历史成本 + 公允价值 |
| 资产减值 | 不计提减值准备 | 需计提 |
| 长期股权投资 | 成本法（无权益法） | 成本法/权益法 |
| 借款费用 | 直接资本化，不区分专门/一般借款 | 需区分 |
| 所得税 | 应付税款法（无递延所得税） | 资产负债表债务法 |
| 财务报表 | 4 张（简化） | 4 张 + 附注（详细） |

## 三、主要会计处理

### 3.1 收入确认

**确认原则**：风险和报酬转移时确认收入

```python
# 销售商品收入确认条件（全部满足）：
# 1. 已将商品所有权上的主要风险和报酬转移给购货方
# 2. 既没有保留通常与所有权相联系的继续管理权，也没有对已售出的商品实施有效控制
# 3. 收入的金额能够可靠地计量
# 4. 相关的经济利益很可能流入企业
# 5. 相关的已发生或将发生的成本能够可靠地计量
```

**本系统实现**：
```python
# 确认销售时
sale.status = SaleStatus.CONFIRMED  # 确认收入
# 同时结转成本
debit: 5401 主营业务成本
  credit: 1405 库存商品
```

### 3.2 存货计价

**计价方法**：实际成本法

| 方法 | 说明 | 本系统支持 |
|------|------|-----------|
| 先进先出法 | 先购入的先发出 | ✅ |
| 加权平均法 | 月末一次加权平均 | ✅ |
| 个别计价法 | 逐一辨认 | ❌ |

**本系统实现**：
```python
# 先进先出法
# 出库时从最早的入库批次开始扣减
inventory_ops.py: deduct_inventory()
```

### 3.3 固定资产折旧

**折旧方法**：年限平均法（直线法）

| 项目 | 规定 |
|------|------|
| 折旧起始 | 投入使用月份的次月起 |
| 折旧停止 | 停止使用月份的次月起 |
| 残值率 | 5%（可自行调整） |

**最低折旧年限**：

| 资产类别 | 最低年限 |
|---------|---------|
| 房屋、建筑物 | 20 年 |
| 飞机、火车、轮船、机器、机械和其他生产设备 | 10 年 |
| 与生产经营活动有关的器具、工具、家具等 | 5 年 |
| 运输工具 | 4 年 |
| 电子设备 | 3 年 |

**本系统实现**：
```python
# 年限平均法
annual_depreciation = (原值 - 残值) / 使用年限
monthly_depreciation = annual_depreciation / 12
```

### 3.4 无形资产摊销

**摊销方法**：直线法

| 项目 | 规定 |
|------|------|
| 摊销起始 | 使用月份的次月起 |
| 使用寿命不确定 | 不摊销 |
| 残值 | 一般为 0 |

### 3.5 借款费用

**资本化条件**：
- 为购建固定资产、无形资产等
- 符合资本化条件时开始资本化
- 达到预定可使用状态时停止资本化

**本系统实现**：
```python
# 借款费用直接资本化，不区分专门借款/一般借款
# 符合小企业会计准则简化处理
```

## 五、主要税法规定

### 5.1 增值税

| 纳税人类型 | 年应税销售额 | 税率 |
|-----------|------------|------|
| 一般纳税人 | > 500 万元 | 13% / 9% / 6% |
| 小规模纳税人 | ≤ 500 万元 | 3%（2023 年起减按 1%） |

**征收率**：
- 小规模纳税人：3%（疫情期间减按 1%）
- 一般纳税人简易计税：3% 或 5%

**进项税额抵扣**：
- 一般纳税人：凭增值税专用发票抵扣
- 小规模纳税人：不抵扣

### 5.2 企业所得税

**税率**：

| 类型 | 税率 |
|------|------|
| 一般企业 | 25% |
| 小微企业（年应纳税所得额 ≤ 300 万） | 5%（2023 年起） |
| 高新技术企业 | 15% |

**小微企业判定**（同时满足）：
- 年应纳税所得额 ≤ 300 万元
- 从业人数 ≤ 300 人
- 资产总额 ≤ 5000 万元

### 5.3 个人所得税（个体工商户/个人独资企业）

| 级数 | 年应纳税所得额 | 税率 | 速算扣除数 |
|------|--------------|------|-----------|
| 1 | ≤ 30,000 | 3% | 0 |
| 2 | 30,000 - 90,000 | 10% | 2,520 |
| 3 | 90,000 - 300,000 | 20% | 16,920 |
| 4 | 300,000 - 500,000 | 30% | 42,920 |
| 5 | > 500,000 | 35% | 65,920 |

### 5.4 其他税费

| 税费 | 计税依据 | 税率 |
|------|---------|------|
| 城市维护建设税 | 增值税额 | 7%（市区）/ 5%（县城）/ 1%（其他） |
| 教育费附加 | 增值税额 | 3% |
| 地方教育附加 | 增值税额 | 2% |
| 印花税 | 合同金额 | 0.03% - 0.1% |
| 房产税 | 房产原值 × 70% × 1.2% 或租金 × 12% | |
| 城镇土地使用税 | 面积 × 单位税额 | |
| 车船税 | 按辆/按吨 | |
