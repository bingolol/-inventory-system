# 📖 小企业会计准则速查

> **法规依据**: 财会〔2011〕17 号《小企业会计准则》
> **时效性**: 现行有效（2013-01-01 施行至今）

---

## 一、准则概述

**适用范围**: 符合《中小企业划型标准规定》（工信部联企业〔2011〕300 号）的小型企业

**与《企业会计准则》的主要区别**:

| 项目 | 小企业会计准则 | 企业会计准则 | 准则条文 |
|------|--------------|------------|---------|
| 计量属性 | 历史成本 | 历史成本 + 公允价值 | 第七条 |
| 资产减值 | 不计提 | 需计提 | 第六条 |
| 长期股权投资 | 成本法 | 成本法/权益法 | 第二十三条 |
| 借款费用 | 直接资本化 | 需区分专门/一般 | 第三十三条 |
| 所得税 | 应付税款法 | 资产负债表债务法 | 第五十三条 |
| 财务报表 | 4 张（简化） | 4 张 + 附注 | 第七十六条 |

## 二、核心计算公式

### 2.1 发票金额计算

**依据**: 《小企业会计准则》第十五条

> 收入应当按照从购买方已收或应收的合同或协议价款确定。

```
不含税金额 = 含税金额 ÷ (1 + 税率)
税额 = 含税金额 - 不含税金额
```

### 2.2 固定资产折旧

**依据**: 《小企业会计准则》第三十条

> 小企业应当按照年限平均法（即直线法）计提折旧。小企业的固定资产由于技术进步等原因，确需加速折旧的，可以采用双倍余额递减法和年数总和法。

**折旧时点**（第三十一条）: 当月增加下月提，当月减少下月停

#### 年限平均法（直线法）

```
月折旧额 = 原值 × (1 - 残值率) ÷ 使用寿命(月)
累计折旧 = 月折旧额 × min(已用月数, 使用寿命)
净值 = 原值 - 累计折旧
```

#### 双倍余额递减法

```
月折旧率 = 2 ÷ 使用寿命(月)
月折旧额 = 期初净值 × 月折旧率
累计折旧 = 逐月累加
净值 = 原值 - 累计折旧（净值不能低于 0）
```

#### 年数总和法

```
年数总和 = n × (n + 1) ÷ 2
月折旧额 = (原值 - 残值) × (剩余年限 / 年数总和) / 12
净值不能低于残值
```

**最低折旧年限**（《企业所得税法实施条例》第六十条）:

| 资产类别 | 最低年限 |
|---------|---------|
| 房屋、建筑物 | 20 年 |
| 机器设备 | 10 年 |
| 器具工具家具 | 5 年 |
| 运输工具 | 4 年 |
| 电子设备 | 3 年 |

### 2.3 无形资产摊销

**依据**: 《小企业会计准则》第四十一条

> 无形资产应当在其使用寿命内采用年限平均法进行摊销。
> 小企业不能可靠估计无形资产使用寿命的，摊销期不得低于 10 年。

```
年摊销额 = 成本 ÷ 使用寿命（年）
月摊销额 = 年摊销额 ÷ 12
```

### 2.4 增值税

**依据**: 《中华人民共和国增值税暂行条例》+ 财政部 税务总局公告 2023 年第 19 号（执行至 2027 年底）

#### 小规模纳税人

```
法定征收率 = 3%
优惠征收率 = 1%（2023-2027）
季度销售额 ≤ 30 万元：免征增值税（含附加税）

不含税销售额 = 含税销售额 ÷ (1 + 1%)
季度销售额 > 30 万元：
  应纳增值税 = 不含税销售额 × 1%
  附加税（减半）：
    城建税 = 增值税 × 7% × 50%
    教育费附加 = 增值税 × 3% × 50%
    地方教育附加 = 增值税 × 2% × 50%
```

#### 一般纳税人

```
税率：13%（货物）/ 9%（运输、建筑）/ 6%（服务）
应纳增值税 = 销项税额 - 进项税额
销项税额 = 不含税销售额 × 税率
进项税额 = 已认证专票税额

附加税（无减半）：
  城建税 = 增值税 × 7%
  教育费附加 = 增值税 × 3%
  地方教育附加 = 增值税 × 2%
```

### 2.5 企业所得税

**依据**: 《中华人民共和国企业所得税法》第五条 + 财政部 税务总局公告 2023 年第 12 号（执行至 2027 年底）

```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 弥补以前年度亏损

【小微企业】（由调用方判断是否符合：利润 ≤ 300 万，人数 ≤ 300，资产 ≤ 5000 万）
应纳税额 = 利润总额 × 25% × 20% = 利润总额 × 5%

【一般企业】
应纳税额 = 利润总额 × 25%
```

## 三、财务报表

### 3.1 资产负债表

**依据**: 《小企业会计准则》第七十六条

```
【资产】
货币资金 = 银行余额 + 现金余额
应收账款 = 未收款销售单总额
预付款项 = 预缴税款（一般纳税人进项＞销项时）
存货 = 库存数量 × 移动加权平均成本
固定资产净值 = 原值 - 累计折旧
无形资产净值 = 原值 - 累计摊销

【负债】
应付账款 = 未付款采购单 + 未付费用 + 未付固定资产 + 未付存货
应交税费 = max(全部销项税 - 可抵扣进项税, 0)（可抵扣进项税仅限已认证的增值税专用发票）
长期借款 = 期初余额录入

【所有者权益】
实收资本 = 期初余额录入
未分配利润 = 期初未分配 + 本期净利润（利润表）
本期净利润 = 营业利润 - 所得税费用

【恒等式】
资产总计 = 负债合计 + 所有者权益合计（容差 ±0.01）
```

### 3.2 利润表

**依据**: 《小企业会计准则》第七十七条

```
营业收入:
  - 一般纳税人 = ∑(不含税销售额)
  - 小规模纳税人 = ∑(含税销售额)
营业成本 = 销售数量 × unit_cost（出库时锁定的移动加权平均成本）
毛利润 = 营业收入 - 营业成本

销售费用
管理费用
财务费用
税金及附加

营业利润 = 毛利润 - 销售费用 - 管理费用 - 财务费用 - 税金及附加
利润总额 = 营业利润（营业外收支暂为 0）
所得税费用 = 利润总额 × 5%（小微企业）或 25%（一般企业）
净利润 = 利润总额 - 所得税费用
```

## 五、会计恒等式校验

**依据**: 《小企业会计准则》第二十一条

### 5.1 资产负债表校验
```
资产 = 负债 + 所有者权益（容差 ±0.01）
```

### 5.2 利润表公式校验（4项）
```
校验1：营业毛利 = 营业收入 - 营业成本
校验2：营业利润 = 营业毛利 - 营业费用（销售+管理+财务+税金及附加）
校验3：利润总额 = 营业利润 + 营业外收入 - 营业外支出
校验4：净利润 = 利润总额 - 所得税费用
```

### 5.3 现金流量表余额校验（2项）
```
校验1：期末余额 = 期初余额 + 净现金流量
校验2：净现金流量 = 经营活动净额 + 投资活动净额 + 筹资活动净额
```

### 5.4 增值税计算校验
```
输入校验：
  - 纳税人类型必须为 small_scale 或 general
  - 营业收入不能为负
  - 进项税额不能为负

输出交叉校验（一般纳税人）：
  - 应纳税额 = 销项税额 - 进项税额

输出交叉校验（小规模纳税人）：
  - 应纳税额 = 不含税销售额 × 1%（季度销售额≤30万时为0）
```

### 5.5 企业所得税计算校验
```
输入校验：
  - 利润不能为负（亏损不缴税）
  - 纳税人类型校验（small_micro/general）

输出交叉校验：
  - 应纳税额 = 应纳税所得额 × 税率 - 减免税额
```

### 5.6 固定资产折旧输入校验
```
  - 原值必须大于 0
  - 残值率必须在 [0, 1] 之间
  - 使用寿命必须大于 0（单位：月）
```

### 5.7 无形资产摊销输入校验
```
  - 原值必须大于 0
  - 使用寿命必须大于 0（单位：月）
  - 累计摊销不超过原值
```

### 5.8 发票金额等式校验
```
不含税金额 + 税额 = 价税合计（容差 ±0.01）
```

## 六、常见会计分录

### 6.1 采购入库

**依据**: 《小企业会计准则》第十三条

```
一般纳税人：
借：库存商品              10,000
借：应交税费-进项税额      1,300
  贷：应付账款              11,300

小规模纳税人：
借：库存商品              11,300
  贷：应付账款              11,300
```

### 6.2 销售出库

**依据**: 《小企业会计准则》第五十九条

```
确认收入：
借：应收账款              22,600
  贷：主营业务收入          20,000
  贷：应交税费-销项税额      2,600

结转成本：
借：主营业务成本          15,000
  贷：库存商品              15,000
```

### 6.3 计提折旧

**依据**: 《小企业会计准则》第三十条

```
借：管理费用-折旧          792
借：销售费用-折旧          396
  贷：累计折旧              1,188

计算：(50,000 - 2,500) ÷ 60 月 = 791.67
```

### 6.4 工资薪酬

**依据**: 《小企业会计准则》第三十九条

```
计提：
借：管理费用-工资          50,000
借：销售费用-工资          30,000
  贷：应付职工薪酬          80,000

发放：
借：应付职工薪酬          80,000
  贷：银行存款              70,000
  贷：应交税费-个人所得税     5,000
  贷：其他应付款-社保        5,000
```

### 6.5 费用报销

**依据**: 《小企业会计准则》第六十七条

```
借：管理费用-房租          5,000
  贷：银行存款              5,000
```

### 6.6 缴纳税金

**依据**: 《小企业会计准则》第四十条
