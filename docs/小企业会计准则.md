# 📖 小企业会计准则速查

> **法规依据**: 财会〔2011〕17 号《小企业会计准则》
> **时效性**: 现行有效（2013-01-01 施行至今）
> **本系统实现**: `backend/accounting_engine.py` 为唯一计算引擎

---

## 一、准则概述

**适用范围**: 符合《中小企业划型标准规定》（工信部联企业〔2011〕300 号）的小型企业

**与《企业会计准则》的主要区别**:

| 项目 | 小企业会计准则 | 企业会计准则 | 准则条文 |
|------|--------------|------------|---------|
| 计量属性 | 历史成本 | 历史成本 + 公允价值 | 第七条 |
| 资产减值 | 不计提 | 需计提 | 第六条 |
| 长期股权投资 | 成本法 | 成本法/权益法 | 第二十三条 |
| 借款费用 | 直接资本化 | 需区分专门/一般 | 第三十三条 |
| 所得税 | 应付税款法 | 资产负债表债务法 | 第五十三条 |
| 财务报表 | 4 张（简化） | 4 张 + 附注 | 第七十六条 |

## 二、核心计算公式

### 2.1 发票金额计算

**依据**: 《小企业会计准则》第十五条

> 收入应当按照从购买方已收或应收的合同或协议价款确定。

```
不含税金额 = 含税金额 ÷ (1 + 税率)
税额 = 含税金额 - 不含税金额
```

**系统实现**: `accounting_engine.py:157-183`

### 2.2 固定资产折旧

**依据**: 《小企业会计准则》第三十条

> 小企业应当按照年限平均法（即直线法）计提折旧。小企业的固定资产由于技术进步等原因，确需加速折旧的，可以采用双倍余额递减法和年数总和法。

**折旧时点**（第三十一条）: 当月增加下月提，当月减少下月停

#### 年限平均法（直线法）

```
年折旧额 = (原值 - 残值) ÷ 使用年限
月折旧额 = 年折旧额 ÷ 12
残值 = 原值 × 残值率（一般 5%）
```

#### 双倍余额递减法

```
年折旧率 = 2 ÷ 使用年限
年折旧额 = 账面净值 × 年折旧率
最后 2 年改直线法: (账面净值 - 残值) ÷ 2
```

#### 年数总和法

```
年数总和 = n × (n + 1) ÷ 2
年折旧额 = (原值 - 残值) × 剩余年限 ÷ 年数总和
```

**最低折旧年限**（《企业所得税法实施条例》第六十条）:

| 资产类别 | 最低年限 |
|---------|---------|
| 房屋、建筑物 | 20 年 |
| 机器设备 | 10 年 |
| 器具工具家具 | 5 年 |
| 运输工具 | 4 年 |
| 电子设备 | 3 年 |

**系统实现**: `accounting_engine.py:218-361`

### 2.3 无形资产摊销

**依据**: 《小企业会计准则》第四十一条

> 无形资产应当在其使用寿命内采用年限平均法进行摊销。
> 小企业不能可靠估计无形资产使用寿命的，摊销期不得低于 10 年。

```
年摊销额 = 成本 ÷ 使用寿命（年）
月摊销额 = 年摊销额 ÷ 12
```

### 2.4 增值税

**依据**: 《中华人民共和国增值税暂行条例》+ 财政部 税务总局公告 2023 年第 19 号（执行至 2027 年底）

#### 小规模纳税人

```
法定征收率 = 3%
优惠征收率 = 1%（2023-2027）
月销售额 ≤ 10 万元：免征增值税

不含税销售额 = 含税销售额 ÷ (1 + 1%)
应纳增值税 = 不含税销售额 × 1%

附加税（小微企业减半）：
  城建税 = 增值税 × 7% × 50%
  教育费附加 = 增值税 × 3% × 50%
  地方教育附加 = 增值税 × 2% × 50%
  月销售额 ≤ 10 万：教育费附加 + 地方教育附加免征
```

#### 一般纳税人

```
税率：13%（货物）/ 9%（运输、建筑）/ 6%（服务）
应纳增值税 = 销项税额 - 进项税额
销项税额 = 不含税销售额 × 税率
进项税额 = 已认证专票税额

附加税（无减半）：
  城建税 = 增值税 × 7%
  教育费附加 = 增值税 × 3%
  地方教育附加 = 增值税 × 2%
```

**系统实现**: `accounting_engine.py:367-440`

### 2.5 企业所得税

**依据**: 《中华人民共和国企业所得税法》第五条 + 财政部 税务总局公告 2023 年第 12 号（执行至 2027 年底）

```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 弥补以前年度亏损

【小微企业】（同时满足：利润 ≤ 300 万，人数 ≤ 300，资产 ≤ 5000 万）
应纳税额 = 利润总额 × 25% × 20% = 利润总额 × 5%

【一般企业】
应纳税额 = 利润总额 × 25%

【高新技术企业】
应纳税额 = 利润总额 × 15%
```

**系统实现**: `accounting_engine.py:446-497`

## 三、财务报表

### 3.1 资产负债表

**依据**: 《小企业会计准则》第七十六条

```
【资产】
货币资金 = 银行余额 + 现金余额
应收账款 = 未收款销售单总额
存货 = 库存数量 × 采购单价
固定资产净值 = 原值 - 累计折旧
无形资产净值 = 原值 - 累计摊销

【负债】
应付账款 = 未付款采购单 + 未付费用 + 未付固定资产 + 未付存货
应交税费 = 销项税额 - 已认证进项税额（≥0）
长期借款 = 期初余额录入

【所有者权益】
实收资本 = 期初余额录入
未分配利润 = 期初未分配 + 本期收入 - 本期成本 - 本期费用 - 折旧 - 摊销 - 所得税

【恒等式】
资产总计 = 负债合计 + 所有者权益合计（容差 ±0.01）
```

**系统实现**: `crud/finance.py: generate_balance_sheet()`

### 3.2 利润表

**依据**: 《小企业会计准则》第七十七条

```
营业收入 = 已完成销售单总额（含税）
营业成本 = 销售数量 × 采购单价
毛利润 = 营业收入 - 营业成本

销售费用 = Expense WHERE functional_category='销售费用'
管理费用 = Expense WHERE functional_category='管理费用'
财务费用 = Expense WHERE functional_category='财务费用'
营业税金及附加 = 增值税附加税

营业利润 = 毛利润 - 税金及附加 - 销售费用 - 管理费用 - 财务费用
利润总额 = 营业利润（营业外收支暂为 0）
所得税费用 = 利润总额 × 5%（小微企业）或 25%（一般企业）
净利润 = 利润总额 - 所得税费用
```

**系统实现**: `crud/finance.py: generate_income_statement()`

## 五、会计恒等式校验

**依据**: 《小企业会计准则》第二十一条

```
资产负债表校验：资产 = 负债 + 所有者权益
期初余额校验：资产 = 负债 + 实收资本 + 未分配利润
利润表校验：净利润 = 利润总额 - 所得税费用
```

**系统实现**:
- `GET /api/accounting-check/balance-sheet` → 资产负债表校验
- `GET /api/accounting-check/income-statement` → 利润表校验
- `GET /api/accounting-check/vat` → 增值税计算校验
- `GET /api/accounting-check/income-tax` → 所得税计算校验
