### 1.2 适用范围

**依据**: 《小企业会计准则》第二条

> 本准则适用于在中华人民共和国境内依法设立的、符合《中小企业划型标准规定》（工信部联企业〔2011〕300 号）所规定的小型企业标准的企业。

**不适用的情形**（第二条）：

1. 股票或债券在市场上公开交易的小企业
2. 企业集团内的母公司和子公司（均指境内依法设立的企业）
3. 符合条件的小企业可以执行本准则，也可以选择执行《企业会计准则》

**执行本准则的小企业公开发行股票或债券的，应当转为执行《企业会计准则》。**

### 1.3 与《企业会计准则》的主要区别

| 项目 | 小企业会计准则 | 企业会计准则 | 法律依据 |
|------|--------------|------------|---------|
| 计量属性 | 历史成本（不使用公允价值） | 历史成本 + 公允价值 | 小企业准则第七条 |
| 资产减值 | 不计提减值准备 | 需计提 | 小企业准则第六条 |
| 长期股权投资 | 成本法（无权益法） | 成本法/权益法 | 小企业准则第二十三条 |
| 借款费用 | 直接资本化，不区分专门/一般借款 | 需区分 | 小企业准则第三十三条 |
| 所得税 | 应付税款法（无递延所得税） | 资产负债表债务法 | 小企业准则第五十三条 |
| 财务报表 | 4 张（简化） | 4 张 + 附注（详细） | 小企业准则第七十六条 |

## 三、主要会计处理与计算公式

### 3.1 收入确认

**依据**: 《小企业会计准则》第五十九条

> 小企业应当在发出商品且收到货款或取得收款权利时确认销售商品收入。

**确认条件**（同时满足）：

1. 已将商品所有权上的主要风险和报酬转移给购货方
2. 既没有保留通常与所有权相联系的继续管理权，也没有对已售出的商品实施有效控制
3. 收入的金额能够可靠地计量
4. 相关的经济利益很可能流入企业
5. 相关的已发生或将发生的成本能够可靠地计量

**计算公式**：

```
销售收入 = 不含税售价 × 数量
含税售价 = 不含税售价 × (1 + 增值税税率)
应收账款 = 含税售价 - 现金折扣（如有）
```

**本系统实现**：

```python
# 确认销售时
sale.status = SaleStatus.CONFIRMED  # 确认收入
# 同时结转成本（准则第六十三条）
debit: 5401 主营业务成本
  credit: 1405 库存商品
```

### 3.2 存货计价

**依据**: 《小企业会计准则》第十四条

> 小企业应当采用先进先出法、加权平均法或者个别计价法确定发出存货的实际成本。

| 方法 | 计算公式 | 适用场景 |
|------|---------|---------|
| 先进先出法 | 先购入的先发出，按入库先后顺序计价 | 价格波动较大 |
| 加权平均法 | 月末一次加权平均 = (月初库存金额 + 本月入库金额) ÷ (月初库存数量 + 本月入库数量) | 价格波动较小 |
| 个别计价法 | 逐一辨认每批发出存货的实际成本 | 贵重物品 |

**加权平均法计算公式**：

```
加权平均单价 = (月初库存金额 + 本月入库金额) ÷ (月初库存数量 + 本月入库数量)
发出存货成本 = 发出数量 × 加权平均单价
期末存货成本 = 期末数量 × 加权平均单价
```

**本系统实现**：

```python
# 先进先出法
# 出库时从最早的入库批次开始扣减（准则第十四条）
inventory_ops.py: deduct_inventory()
```

### 3.3 固定资产折旧

**依据**: 《小企业会计准则》第三十条至第三十一条

> 小企业应当按照年限平均法（即直线法）计提折旧。
> 小企业的固定资产由于技术进步等原因，确需加速折旧的，可以采用双倍余额递减法和年数总和法。

**折旧计算公式**：

```
年折旧额 = (原值 - 预计净残值) ÷ 预计使用寿命（年）
月折旧额 = 年折旧额 ÷ 12
```

**加速折旧公式**：

```
双倍余额递减法：
年折旧率 = 2 ÷ 预计使用寿命（年）× 100%
年折旧额 = 固定资产账面净值 × 年折旧率
最后两年改用直线法：(账面净值 - 预计净残值) ÷ 2

年数总和法：
年折旧率 = 尚可使用年限 ÷ 年数总和 × 100%
年折旧额 = (原值 - 预计净残值) × 年折旧率
```

**折旧时点**（第三十一条）：

- 当月增加的固定资产，**当月不计提折旧**，从下月起计提
- 当月减少的固定资产，**当月仍计提折旧**，从下月起不计提

**最低折旧年限**（依据《企业所得税法实施条例》第六十条）：

| 资产类别 | 最低年限 | 法律依据 |
|---------|---------|---------|
| 房屋、建筑物 | 20 年 | 企业所得税法实施条例第六十条第一项 |
| 飞机、火车、轮船、机器、机械和其他生产设备 | 10 年 | 企业所得税法实施条例第六十条第二项 |
| 与生产经营活动有关的器具、工具、家具等 | 5 年 | 企业所得税法实施条例第六十条第三项 |
| 飞机、火车、轮船以外的运输工具 | 4 年 | 企业所得税法实施条例第六十条第四项 |
| 电子设备 | 3 年 | 企业所得税法实施条例第六十条第五项 |

**预计净残值**：

- 《小企业会计准则》未规定具体残值率
- 《企业所得税法实施条例》第五十九条规定：企业应当根据固定资产的性质和使用情况，合理确定固定资产的预计净残值
- **实务中一般按 5% 计算**（税务机关认可的惯例）

**本系统实现**：

```python
# 年限平均法（准则第三十条）
annual_depreciation = (原值 - 残值) / 使用年限
monthly_depreciation = annual_depreciation / 12
```

### 3.4 无形资产摊销

**依据**: 《小企业会计准则》第四十一条

> 无形资产应当在其使用寿命内采用年限平均法进行摊销，根据其受益对象计入相关资产成本或者当期损益。

**摊销计算公式**：

```
年摊销额 = 无形资产成本 ÷ 使用寿命（年）
月摊销额 = 年摊销额 ÷ 12
```

**摊销时点**（第四十一条）：

- 摊销期自其**可供使用时**开始至**停止使用或出售时**止

**使用寿命不确定的无形资产**（第四十一条）：

> 小企业不能可靠估计无形资产使用寿命的，摊销期**不得低于 10 年**。

**本系统实现**：

```python
# 年限平均法（准则第四十一条）
annual_amortization = 无形资产成本 / 使用寿命（年）
monthly_amortization = annual_amortization / 12
```

### 3.5 长期待摊费用

**依据**: 《小企业会计准则》第四十三条

> 长期待摊费用应当在其摊销期限内采用年限平均法进行摊销，根据其受益对象计入相关资产的成本或者管理费用，并冲减长期待摊费用。

**包含内容**：

1. 已提足折旧的固定资产的改建支出
2. 经营租入固定资产的改建支出
3. 固定资产的大修理支出（同时满足：修理支出达到取得固定资产时的计税基础 50% 以上；修理后固定资产的使用寿命延长 2 年以上）
4. 其他长期待摊费用

**摊销公式**：

```
月摊销额 = 长期待摊费用总额 ÷ 摊销期限（月）
```

### 3.6 借款费用

**依据**: 《小企业会计准则》第三十三条

> 小企业为购建固定资产在竣工决算前发生的借款费用，应当计入固定资产的成本。

**资本化条件**：

- 为购建固定资产、无形资产等
- 符合资本化条件时开始资本化
- 达到预定可使用状态时停止资本化

**计算公式**：

```
资本化金额 = 借款本金 × 借款利率 × 资本化期间
费用化金额 = 借款利息 - 资本化金额
```

**本系统实现**：

```python
# 借款费用直接资本化，不区分专门借款/一般借款（准则第三十三条）
# 符合小企业会计准则简化处理
```

### 3.7 坏账损失

**依据**: 《小企业会计准则》第十条

> 小企业应收及预付款项符合下列条件之一的，减除可收回的金额后确认的无法收回的应收及预付款项，作为坏账损失：
> 1. 债务人依法宣告破产、关闭、解散、被撤销，或者被依法注销、吊销营业执照，其清算财产不足清偿的
> 2. 债务人死亡，或者依法被宣告失踪、死亡，其财产或者遗产不足清偿的
> 3. 债务人逾期 3 年以上未清偿，且有确凿证据证明已无力清偿债务的
> 4. 与债务人达成债务重组协议或法院批准破产重整计划后，无法追偿的
> 5. 因自然灾害、战争等不可抗力导致无法收回的

**处理方式**：

```
借：营业外支出
  贷：应收账款
```

**注意**：小企业会计准则**不计提坏账准备**，直接确认损失。

## 四、财务报表

### 4.1 资产负债表

**依据**: 《小企业会计准则》第七十六条

**格式**: 报告式（上下结构）

**资产排列**: 流动性从高到低

**编制公式**：

```
资产 = 负债 + 所有者权益
流动资产合计 = 货币资金 + 短期投资 + 应收票据 + 应收账款 + 预付账款 + 其他应收款 + 存货 + 其他流动资产
非流动资产合计 = 长期债券投资 + 长期股权投资 + 固定资产净值 + 在建工程 + 无形资产净值 + 长期待摊费用 + 其他非流动资产
资产总计 = 流动资产合计 + 非流动资产合计
流动负债合计 = 短期借款 + 应付票据 + 应付账款 + 预收账款 + 应付职工薪酬 + 应交税费 + 应付利息 + 应付股利 + 其他应付款 + 其他流动负债
非流动负债合计 = 长期借款 + 长期应付款 + 递延收益 + 其他非流动负债
负债合计 = 流动负债合计 + 非流动负债合计
所有者权益合计 = 实收资本 + 资本公积 + 盈余公积 + 未分配利润
负债和所有者权益总计 = 负债合计 + 所有者权益合计
```

### 4.2 利润表

**依据**: 《小企业会计准则》第七十七条

**格式**: 多步式

**编制公式**：

```
营业收入 = 主营业务收入 + 其他业务收入
营业成本 = 主营业务成本 + 其他业务成本
营业利润 = 营业收入 - 营业成本 - 营业税金及附加 - 销售费用 - 管理费用 - 财务费用 + 投资收益
利润总额 = 营业利润 + 营业外收入 - 营业外支出
净利润 = 利润总额 - 所得税费用
```

### 4.3 现金流量表

**依据**: 《小企业会计准则》第七十八条

**编制方法**: 直接法

**编制公式**：

```
经营活动现金流量净额 = 经营活动现金流入 - 经营活动现金流出
投资活动现金流量净额 = 投资活动现金流入 - 投资活动现金流出
筹资活动现金流量净额 = 筹资活动现金流入 - 筹资活动现金流出
现金及现金等价物净增加额 = 经营活动现金流量净额 + 投资活动现金流量净额 + 筹资活动现金流量净额
期末现金及现金等价物余额 = 期初现金及现金等价物余额 + 现金及现金等价物净增加额
```

### 4.4 附注

**依据**: 《小企业会计准则》第七十九条

> 小企业应当遵循附注披露的最低要求编制财务报表附注。

**注意**: 小企业可以不编制附注（简化要求），但建议至少披露重要事项。

## 五、主要税法规定

### 5.1 增值税

**依据**: 《中华人民共和国增值税暂行条例》

| 纳税人类型 | 年应税销售额 | 税率 | 法律依据 |
|-----------|------------|------|---------|
| 一般纳税人 | > 500 万元 | 13% / 9% / 6% | 增值税暂行条例第二条 |
| 小规模纳税人 | ≤ 500 万元 | 3%（减按 1%） | 增值税暂行条例第十二条 |

**小规模纳税人优惠政策**（财政部 税务总局公告 2023 年第 19 号，执行至 **2027 年 12 月 31 日**）：

| 优惠内容 | 条件 | 政策依据 |
|---------|------|---------|
| 免征增值税 | 月销售额 ≤ 10 万元（季度 ≤ 30 万元） | 财政部 税务总局公告 2023 年第 19 号第一条 |
| 减按 1% 征收率 | 适用 3% 征收率的应税销售收入 | 财政部 税务总局公告 2023 年第 19 号第二条 |

**计算公式**：

```
应纳税额 = 不含税销售额 × 征收率
不含税销售额 = 含税销售额 ÷ (1 + 征收率)
```

**进项税额**：

- 一般纳税人：凭增值税专用发票抵扣
- 小规模纳税人：不抵扣进项税额

### 5.2 企业所得税

**依据**: 《中华人民共和国企业所得税法》

**税率**（企业所得税法第四条）：

| 类型 | 税率 | 法律依据 |
|------|------|---------|
| 一般企业 | 25% | 企业所得税法第四条第一款 |
| 小型微利企业 | 实际税率 5% | 财政部 税务总局公告 2023 年第 12 号 |
| 高新技术企业 | 15% | 企业所得税法第二十八条 |

**小型微利企业判定条件**（同时满足）：

1. 年应纳税所得额 ≤ 300 万元
2. 从业人数 ≤ 300 人
3. 资产总额 ≤ 5000 万元
4. 从事国家非限制和禁止行业

**小微企业所得税优惠政策**（财政部 税务总局公告 2023 年第 12 号，执行至 **2027 年 12 月 31 日**）：

> 对小型微利企业减按 25% 计算应纳税所得额，按 20% 的税率缴纳企业所得税。

**计算公式**：

```
应纳税所得额 = 收入总额 - 不征税收入 - 免税收入 - 各项扣除 - 允许弥补的以前年度亏损
应纳税额 = 应纳税所得额 × 税率
小微企业实际应纳税额 = 应纳税所得额 × 25% × 20% = 应纳税所得额 × 5%
```

**计算示例**：

```
某小微企业年应纳税所得额为 200 万元
应纳税额 = 200 × 25% × 20% = 10 万元（实际税率 5%）
若按一般税率 25% 计算：200 × 25% = 50 万元
减免税额 = 50 - 10 = 40 万元
```

### 5.3 个人所得税（个体工商户/个人独资企业）

**依据**: 《中华人民共和国个人所得税法》

**经营所得税率表**（个人所得税法第三条第二款）：

| 级数 | 年应纳税所得额 | 税率 | 速算扣除数 |
|------|--------------|------|-----------|
| 1 | ≤ 30,000 | 3% | 0 |
| 2 | 30,000 - 90,000 | 10% | 2,520 |
| 3 | 90,000 - 300,000 | 20% | 16,920 |
| 4 | 300,000 - 500,000 | 30% | 42,920 |
| 5 | > 500,000 | 35% | 65,920 |

**计算公式**：

```
应纳税额 = 应纳税所得额 × 税率 - 速算扣除数
应纳税所得额 = 收入总额 - 成本 - 费用 - 损失
```

**个体工商户优惠政策**（财政部 税务总局公告 2023 年第 12 号，执行至 **2027 年 12 月 31 日**）：

> 对个体工商户年应纳税所得额不超过 200 万元的部分，减半征收个人所得税。

### 5.4 附加税费

**依据**: 《中华人民共和国城市维护建设税法》《征收教育费附加的暂行规定》

| 税费 | 计税依据 | 税率 | 法律依据 |
|------|---------|------|---------|
| 城市维护建设税 | 增值税额 | 7%（市区）/ 5%（县城）/ 1%（其他） | 城市维护建设税法第四条 |
| 教育费附加 | 增值税额 | 3% | 征收教育费附加的暂行规定第三条 |
| 地方教育附加 | 增值税额 | 2% | 各省实施办法 |

**小微企业优惠政策**（财政部 税务总局公告 2023 年第 12 号，执行至 **2027 年 12 月 31 日**）：

> 对增值税小规模纳税人、小型微利企业和个体工商户减半征收资源税（不含水资源税）、城市维护建设税、房产税、城镇土地使用税、印花税（不含证券交易印花税）、耕地占用税和教育费附加、地方教育附加。

**计算公式**：

```
城市维护建设税 = 增值税额 × 税率 × 50%（小微企业减半）
教育费附加 = 增值税额 × 3% × 50%（小微企业减半）
地方教育附加 = 增值税额 × 2% × 50%（小微企业减半）
```

### 5.5 其他税费

| 税费 | 计税依据 | 税率 | 法律依据 |
|------|---------|------|---------|
| 印花税 | 合同金额 | 0.03% - 0.1% | 《中华人民共和国印花税法》 |
| 房产税 | 房产原值 × 70% × 1.2% 或租金 × 12% | | 《中华人民共和国房产税暂行条例》 |
| 城镇土地使用税 | 面积 × 单位税额 | | 《中华人民共和国城镇土地使用税暂行条例》 |
| 车船税 | 按辆/按吨 | | 《中华人民共和国车船税法》 |
