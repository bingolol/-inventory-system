## 交付目标

### T1：跨表勾稽 — 利润表 ↔ 资产负债表

**断言**: `利润表.净利润 == 资产负债表.期末未分配利润 - 期初未分配利润`

场景：采购→销售→费用→收款→付款 完整闭环后校验。

### T2：成本计算 — COGS 公式验证

**断言**: `SaleItem.cost == InventoryEngine 计算的加权平均成本 × 销量`

场景：两批不同单价采购（¥10 + ¥12）→ 销售 → COGS = 当时的 avg_cost × 销量。

### T3：库存真相源 — StockMove ↔ Inventory 对账

**断言**: `∑(StockMove.inbound) - ∑(StockMove.outbound) == Inventory.quantity`

场景：采购入库 + 销售出库 + 红冲后，逐产品核对。

### T4：库存估值 — avg_cost × qty = total_value

**断言**: `Inventory.average_cost * Inventory.quantity == Inventory.total_value`

场景：每次库存变动后验证，允许 rounding ±0.01。

### T5：应收对账 — 应收账款 ↔ 未收销售单

**断言**: `资产负债表.应收账款 == ∑(销售单.未付金额)`

场景：多笔销售 + 部分收款后验证。

### T6：附加税公式 — surcharge_total = vat_payable × 12%

**断言**: `城建税 + 教育费附加 + 地方教育附加 == 增值税应缴 × 12%`

场景：一般纳税人月度增值税计算后验证。
