## 数据层级 (L1–L4)

所有数据字段按"数字源原则"分四层：

| 层级 | 含义 | 示例 | 规则 |
|------|------|------|------|
| **L1** | 外部输入 | `Invoice.amount`、`PurchaseOrder.total_price_l1` | 仅 API 边界写入 |
| **L2** | 引擎计算—内部真相源 | `StockMove.unit_cost_l2`（原始凭证）、`CostEngine.weighted_average()`（计算）| Writer 唯一，L4 不得回读为计算输入 |
| **L3** | 政策配置 | `Account.taxpayer_type_l3`、`Product.track_inventory_l3` | 创建时配置，引擎做分支判断 |
| **L4** | 派生汇总—性能缓存 | `Inventory.average_cost_l4`、`SaleItem.unit_cost_l2`(快照) | 引擎自动维护，仅供报表/API 查询 |

追溯链: L1 发票 → L2 StockMove(原始凭证) → L2 CostEngine(计算) → L4 缓存 + 凭证 COGS
