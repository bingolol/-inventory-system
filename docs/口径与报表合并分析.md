# 口径与报表合并分析

## 一、当前报表口径对照

| 报表 | 路由 | 当前口径 | 真相源 |
|---|---|---|---|
| 增值税季度/月度报表 | `/api/tax-report` | 税务口径（发票说话） | `Invoice` + `SaleOrder`（未开票收入） |
| 企业所得税报表 | `/api/income-tax-report` | 税务口径（发票说话） | `Invoice` + `Expense` |
| 企业所得税预缴表 | `generate_income_tax_prepayment` | 混合口径 | 销项发票 + `SaleItem.unit_cost` + `Expense` + VAT附加 |
| 资产负债表 | `/api/financial-reports/balance-sheet` | 会计准则口径 | 总账 `AccountMoveLine` + 期初余额 |
| 利润表 | `/api/financial-reports/income-statement` | 会计准则口径 | 总账 `AccountMoveLine` |
| 小企业会计准则报表 | `/api/financial-reports/cwbb-xqykjzz` | 会计准则口径 | 复用资产负债表 + 利润表 + 现金流量表 |
| 期初余额 | `/api/opening-balances` | 外部录入（L1） | `OpeningBalance` 模型 |

## 二、税务报表切到会计准则口径

### 2.1 增值税报表保持发票口径

增值税申报以**发票**和**未开票收入**为法定依据，不能改为会计准则口径。保持现有 `/api/tax-report` 不变。

### 2.2 企业所得税改为会计准则口径

企业所得税预缴/年度申报的营业收入、营业成本、利润总额，应与《小企业会计准则》利润表一致。

**调整方案：**

| 项目 | 旧取数（发票口径） | 新取数（会计准则口径） |
|---|---|---|
| 营业收入 | 销项发票不含税金额 | 总账 `6001` 贷方 + `6051` 贷方 |
| 营业成本 | 进项发票不含税金额 | 总账 `6401` 借方 |
| 税金及附加 | 无 | 总账 `6403` 及明细借方 |
| 期间费用 | 有票 `Expense` | 总账 `6601+6602+6603` 借方 |
| 营业外收入 | 仅 VAT 减免（6301） | 总账 `6301+6111` 贷方 |
| 营业外支出 | 无 | 总账 `6701+6711` 借方 |
| 利润总额 | 发票收入 − 发票成本 − 有票费用 | 利润表 `gross_profit_total` |
| 应纳税所得额 | 利润总额 | 利润总额（暂不处理纳税调整） |

**实现方式：** 直接复用 `crud.finance.generate_income_statement` 的结果，避免双轨计算。

## 三、资产负债表 / 期初余额 / 税务报表 合并可能性

### 3.1 资产负债表与期初余额：不能合并

- **期初余额**是时点输入（L1 外部录入），用于初始化总账。
- **资产负债表**是任意时点的派生报表（L4），已从总账计算。
- 关系：资产负债表期末数 = 期初余额 + 本期总账发生额。二者是输入与输出的关系，保留独立入口。

### 3.2 资产负债表与小企业会计准则报表：已实质合并

`generate_cwbb_xqykjzz` 已直接调用 `generate_balance_sheet` 生成会小企01表，无需额外合并。

### 3.3 税务报表与财务报表：不能合并，但需对账

- 用途不同：财务报表给股东/管理层，税务报表给税务局。
- 数据不同：增值税必须按发票；所得税按会计准则利润。
- 但**利润总额应与利润表一致**，否则企业所得税申报会出现会计利润与申报利润不一致。

### 3.4 可合并的重复接口

建议合并以下重复或接近的路由：

| 保留 | 建议废弃/迁移 | 理由 |
|---|---|---|
| `/api/financial-reports/balance-sheet` | `/api/finance/accounts/chart` 中余额展示 | 报表层更完整 |
| `/api/financial-reports/income-statement` | `/api/reports/profit` | 前者是准则口径，后者是经营口径，应更名避免混淆 |
| `/api/financial-reports/financial-summary` | `/api/reports/overview` 中财务部分 | 统一为“财务报表概览” |
