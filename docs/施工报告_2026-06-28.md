## 一、修复问题清单

### 1.1 资产负债表不平衡 — 4层根因

| 层次 | 问题 | 修复 |
|------|------|------|
| L1 | `sum([])` 返回 `int 0`，`0.quantize(Q2)` 崩溃 | 所有 `= sum(...)` 加 `_d(sum(...))` 包裹 |
| L2 | SQLite 存 DateTime 带时间，`date` 对象比较 `<=` 字母序失败 | `query_date` 改 `datetime`，加 `query_end`(23:59:59) |
| L3 | BS 用 `Inventory` 快照表（当前值）而非按日期过滤 | 加 `StockMove.move_date` 列，BS 从 StockMove 按 `move_date` 过滤 |
| L4 | StockMove 无业务日期，`created_at` 全是当前时间 | `InventoryEngine._get_move_date()` 从源单据取 `purchase_date`/`sale_date` |
