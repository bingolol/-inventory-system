## 📦 商品管理

### 功能说明

- 商品信息 CRUD
- 商品分类管理
- SKU 编码管理
- 库存预警设置

### 业务规则

1. **删除约束**: 存在采购或销售记录的商品不能删除
2. **SKU 唯一**: 同一账本下 SKU 不能重复
3. **库存跟踪**: `track_inventory` 字段控制是否跟踪库存

### 数据模型

```python
class Product:
    id: int
    account_id: int
    name: str
    sku: str
    category: str
    unit: str
    purchase_price: Decimal
    sale_price: Decimal
    min_stock: int
    track_inventory: bool
    description: str
```

## 📊 库存管理

### 功能说明

- 库存数量查询
- 库存预警（低于 min_stock）
- 库存盘点调整
- 库存变动日志

### 库存联动规则

| 操作 | 库存变化 |
|------|----------|
| 创建采购单 (track_inventory=true) | +quantity |
| 取消/删除采购单 (已完成) | -quantity |
| 创建销售单 (deduct_inventory=true, track_inventory=true) | -quantity |
| 取消/删除销售单 | +quantity |
| 库存盘点调整 | 设为指定值 |

### 业务规则

1. **库存不足校验**: 扣减时检查库存是否充足
2. **track_inventory**: False 的商品跳过库存操作
3. **原子操作**: 库存变动记录到 OperationLog

## 🛒 采购管理

### 功能说明

- 采购单 CRUD
- 采购入库联动
- 采购单状态管理
- 付款状态跟踪

### 业务规则

1. **入库联动**: 创建采购单自动增加库存
2. **回补联动**: 取消/删除已完成采购单自动减少库存
3. **状态机**: pending → completed → cancelled
4. **订单号格式**: CG{年}—{月}-{日}-{时}:{分}-{序号}

### 数据模型

```python
class PurchaseOrder:
    id: int
    account_id: int
    order_no: str
    supplier_id: int
    tax_amount: Decimal
    payment_method: str  # 'company' | 'private_advance'
    payment_status: str  # 'paid' | 'unpaid' | 'partial'
    status: str          # 'pending' | 'completed' | 'cancelled'
    total_price: Decimal  # 价税合计
    notes: str
    items: List[PurchaseItem]
```

## 🧾 发票管理

### 功能说明

- 进项/销项发票管理
- 普票/专票区分
- 专票认证状态
- 税额自动计算

### 业务规则

1. **发票号码唯一**: 同一账本下发票号码不能重复
2. **税额计算**: `tax_amount = amount_with_tax × tax_rate ÷ (1 + tax_rate)`
3. **认证状态**: n_a (不适用) → pending (待认证) → certified (已认证)
4. **商品明细**: 每张发票可关联多行商品（InvoiceItem），自动计算行小计
5. **自动生成订单**: 销项发票可自动生成销售单（扣库存），进项发票可自动生成采购单（入库）

### 新字段

- **seller_name / buyer_name**: 发票的销方/买方名称（从 AI 识别结果写入）
- **tax_amount**: SaleOrder / PurchaseOrder 新增税额字段，记录订单的增值税额
- **InvoiceItem**: 发票商品明细行，记录每行商品、数量、单价、税率

### AI 快捷录入

- `POST /api/invoices/quick` — 统一入口，支持：
  - 传 `items[]` 自动保存商品明细
  - `sale_order_action=auto_create` → 自动生成已完成销售单并扣库存
  - `purchase_order_action=auto_create` → 自动生成已完成采购单并入库
  - `sale_order_action=link_existing` → 关联已有销售单
  - `purchase_order_action=link_existing` → 关联已有采购单

### 税率选项

| 税率 | 说明 |
|------|------|
| 0.01 | 1% (小规模优惠) |
| 0.03 | 3% |
| 0.06 | 6% (服务业) |
| 0.09 | 9% (交通运输) |
| 0.13 | 13% (一般商品) |
