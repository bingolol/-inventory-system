# 期初余额系统使用指南

## 系统概述

期初余额系统是进销存管理系统的重要组成部分，专门用于处理一般纳税人企业的期初财务数据设置和财务报表生成。系统支持完整的资产负债表、利润表生成，并提供财务健康度分析功能。

## 功能特性

### ✅ 核心功能
- **期初余额管理**：支持资产、负债、权益类科目的期初余额录入
- **资产负债表生成**：自动生成符合会计准则的资产负债表
- **利润表生成**：基于业务数据自动生成利润表
- **财务健康度分析**：提供资产负债率、流动比率等关键财务指标
- **数据平衡检查**：自动验证资产负债表平衡关系
- **历史记录管理**：支持期初余额的历史记录查看和编辑

### ✅ 技术特性
- **数据安全**：基于账户隔离，确保数据安全
- **实时计算**：报表数据实时计算，保证准确性
- **用户友好**：直观的界面设计和操作指引
- **扩展性强**：支持后续功能扩展和定制

## 快速开始

### 1. 系统部署

```bash
# 克隆项目后，执行部署脚本
cd inventory-system
chmod +x deploy_opening_balance.sh
./deploy_opening_balance.sh
```

### 2. 访问系统

- 系统地址：http://localhost:8000
- API文档：http://localhost:8000/docs

### 3. 设置期初余额

1. 登录系统后，点击左侧菜单"期初余额"
2. 选择期初日期（通常为会计年度开始日期）
3. 填写各项财务数据：
   - **资产类**：现金余额、银行存款、应收账款、库存价值
   - **负债类**：应付账款、应交税费
   - **权益类**：未分配利润
4. 系统会自动检查资产负债表平衡
5. 点击"保存期初余额"完成设置

## 详细功能说明

### 期初余额管理

#### 数据录入
- **现金余额**：企业库存现金金额
- **银行存款**：企业在各银行的存款总额
- **应收账款**：客户欠款总额
- **库存价值**：库存商品的总价值
- **应付账款**：欠供应商的款项总额
- **应交税费**：应缴纳的各项税费
- **未分配利润**：企业累计未分配的利润

#### 平衡检查
系统会自动验证会计恒等式：资产 = 负债 + 所有者权益
- ✅ 平衡：数据显示绿色，可以保存
- ❌ 不平衡：数据显示红色，需要调整数据

#### 历史记录
- 支持查看所有历史期初余额记录
- 支持编辑和删除操作
- 支持加载最新记录功能

### 财务报表

#### 资产负债表
- **报表日期**：可选择任意日期生成报表
- **资产部分**：显示流动资产各项目
- **负债部分**：显示流动负债各项目
- **权益部分**：显示所有者权益
- **平衡检查**：自动验证报表平衡性

#### 利润表
- **期间选择**：可自定义报表期间
- **收入成本**：显示营业收入和营业成本
- **利润计算**：自动计算毛利润、营业利润、净利润
- **盈利分析**：显示盈利状态和关键指标

#### 财务汇总
- **三栏展示**：资产、负债、权益分别展示
- **健康度分析**：提供关键财务比率分析
- **状态监控**：显示期初余额设置状态

## 使用场景

### 场景1：新企业初始化
1. 在企业成立时设置期初余额
2. 录入初始投资、初始库存等数据
3. 生成初始资产负债表

### 场景2：年度结转
1. 在新会计年度开始时更新期初余额
2. 基于上年期末数据设置新年期初
3. 确保财务数据连续性

### 场景3：月度财务分析
1. 每月生成资产负债表和利润表
2. 分析企业财务状况变化
3. 为经营决策提供数据支持

## 注意事项

### 数据准确性
- 期初余额数据直接影响后续所有财务报表
- 请确保数据来源可靠、计算准确
- 建议在专业会计人员指导下录入

### 平衡关系
- 必须满足：资产 = 负债 + 所有者权益
- 系统会自动检查，不平衡时无法保存
- 如数据不平衡，请仔细核对各项数据

### 时间选择
- 期初日期通常为会计期间开始日
- 一个会计期间只能有一个期初余额
- 修改期初余额会影响后续所有报表

## 常见问题

### Q1: 为什么无法保存期初余额？
**A**: 可能原因：
- 资产负债表不平衡（最常见）
- 日期格式不正确
- 必填字段未填写
- 该日期已存在期初余额记录

### Q2: 如何修改已保存的期初余额？
**A**: 
1. 在期初余额页面查看历史记录
2. 点击对应记录的"编辑"按钮
3. 修改数据后保存
4. 系统会重新检查平衡关系

### Q3: 报表数据不准确怎么办？
**A**:
1. 检查期初余额数据是否正确
2. 确认业务数据录入完整
3. 检查报表期间设置是否合理
4. 如问题持续，请联系技术支持

### Q4: 支持哪些财务报表？
**A**: 目前支持：
- 资产负债表
- 利润表
- 财务汇总分析
后续将支持现金流量表等更多报表
