## 🔗 快速查找

### 按功能查找

| 我想找... | 文件位置 |
|-----------|----------|
| 添加新 API 端点 | `backend/routers/` |
| 添加新写操作 | `backend/commands/` |
| 添加新查询 | `backend/crud/` |
| 添加新模型 | `backend/models.py` |
| 添加新枚举 | `backend/enums.py` |
| 添加新页面 | `frontend/src/views/` |
| 添加新测试 | `tests/unit/` 或 `tests/integration/` |
| 选 AI 技能 | `.agents/skills/`（指南见 [agent_skills.md](./agent_skills.md)） |
| 查看错误码 | `backend/errors.py` |
| 查看业务规则 | `backend/domain/` |

### 按场景查找

| 场景 | 需要修改的文件 |
|------|----------------|
| 新增商品字段 | `models.py` → `schemas/` → `commands/product_commands.py` → `crud/products.py` → `routers/products.py` → 前端 |
| 新增采购逻辑 | `commands/purchase_commands.py` → `crud/orders.py` → `routers/purchases.py` |
| 新增报表 | `crud/finance.py` 或 `crud/reports.py` → `routers/financial_reports.py` |
| 新增枚举 | `enums.py` → `schemas/` → 前端 |
