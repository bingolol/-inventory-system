# 进销存管理系统

## Register
product

## Target Users
中国中小企业主、个体经营者、财务人员。无技术背景，需要一套简单直观的业务管理工具。

## Product Purpose
一站式业务管理平台，涵盖进货（采购）、销售、库存、财务核算和税务申报核心环节。让中小企业主在一个系统里完成从采购到收款的完整业务闭环，实时了解经营状况。

## Core Modules
- 商品管理 — 商品信息维护、SKU 管理
- 采购管理 — 采购订单、入库、供应商管理
- 销售管理 — 销售订单、出库、客户管理
- 库存管理 — 库存盘点、预警、调拨
- 财务管理 — 财务总览、银行账户、现金流、费用管理
- 财务报表 — 资产负债表、利润表
- 税务管理 — 增值税报表、企业所得税报表、发票管理
- 会计核算 — 会计凭证、试算平衡表、往来账龄

## Brand Personality
专业、清晰、高效。不花哨，但精致。数据密集但易于阅读。面向中小企业主，界面语言必须全部中文，避免会计专业术语在前端直接展示。

## Design Principles
1. 数据一目了然 — 核心指标放在显眼位置，不翻页就能看到关键数据
2. 计算过程可见 — 财务数据展示计算公式，让用户理解数字来源
3. 杂志风格 — 卡片式布局、充足留白、圆角、柔和阴影
4. 色彩克制 — 深蓝主色调 #4f6ef7，功能色绿/黄/红/灰
5. 深色侧栏 — 现代 SaaS 标配，区分导航区和内容区
6. 状态可视化 — 统一药丸标签表示状态（成功/警告/危险/信息）

## Anti-References
- 传统 ERP 的密集表格和灰色界面
- Element Plus 默认的蓝白 admin 风格
- 一行行枯燥的数字列表没有视觉层次
